# Week 4 – Intermediate JavaScript

This week focuses on strengthening core JavaScript concepts by building dynamic web applications. The goal is to gain hands-on experience with arrays, objects, higher-order array methods, template literals, and DOM manipulation while creating interactive user interfaces.

## Topics Covered

* Arrays
* Arrays of Objects
* Object Properties
* `forEach()`
* `map()`
* `filter()`
* Arrow Functions
* Template Literals
* DOM Selection & Manipulation
* Dynamic Content Generation
* Event Listeners
* Search Functionality
* Filter Functionality

## Projects

### Project 1 – Student Directory 

A simple Student Directory application that dynamically displays student information stored in an array of objects. Built using JavaScript to generate HTML elements and render data directly onto the webpage.

**Concepts Practiced:**

* Arrays of Objects
* Object Property Access
* `forEach()`
* Template Literals
* DOM Manipulation
* Dynamic HTML Generation
